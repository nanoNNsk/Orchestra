🎻 Orchestra — คู่มือใช้งาน

คุยกับ "บริษัท AI" ทั้งบริษัท ไม่ใช่บอตตัวเดียว

💡 คลิกที่กล่องคำสั่งสีดำเพื่อ คัดลอก ได้เลย

อยากเริ่มเร็วสุด? ทำ 3 ขั้นนี้พอ — เปิดแชทได้ใน 1 นาที
1 · เข้าโฟลเดอร์โปรเจกต์
cd path/to/orchestracopy
2 · ติดตั้ง (ทำครั้งเดียว)
npm installcopy
3 · เปิดแชท
npm startcopy

แล้วเปิดเบราว์เซอร์ไปที่ http://localhost:4567 — จบ ✅

0สิ่งที่ต้องมีก่อน

  • Node.js เวอร์ชัน 20 ขึ้นไป — เช็คด้วย node -v
  • Codex CLI ที่ล็อกอินแล้ว (ตัว AI ขับด้วยอันนี้ ไม่ต้องใช้ API key) — เช็คด้วย codex login status ควรขึ้น Logged in
  • Python 3 — ใช้แค่ตอนตรวจ roster (npm run validate)
ถ้า codex login status ไม่ขึ้นว่า logged in ให้รัน codex login ก่อน ไม่งั้นแชทจะตอบไม่ได้

1เปิดแชท (ทางหลัก)

npm startcopy

จะขึ้นข้อความ Orchestra chat → http://localhost:4567 แล้วเปิดลิงก์นั้นในเบราว์เซอร์

วิธีใช้หน้าแชท:

  • พิมพ์บอก CEO ว่าต้องการอะไร แล้วกด Send — ตอบไทยหรืออังกฤษก็ได้
  • จะเห็น trace ว่างานถูกส่งไปแผนกไหน/ใครทำ ก่อนคำตอบสรุป
  • ปุ่ม 🏛 Company (มุมขวาบน) เปิดดูผังบริษัทสดๆ — กดไล่ แผนก → role → skill ได้
  • แชทถูกเซฟอัตโนมัติ refresh แล้วไม่หาย · ปุ่ม Clear ล้างเฉพาะหน้าจอคุณ (บริษัทยังจำงานเก่าได้)
  • ถ้ามีงานที่ไม่มีใครทำได้ บริษัทจะ จ้างคนใหม่ ให้เห็น 🆕 เด้งใน panel
ปิด server: กลับไปที่หน้าต่าง terminal แล้วกด Ctrl + C · แต่ละคำถามใช้เวลา ~20–60 วินาที (ยิง AI หลายรอบ)

2คำสั่งทั้งหมด

คำสั่งทำอะไร
npm startเปิดหน้าแชทในเบราว์เซอร์ (ทางหลัก)
npm run statusดูผังบริษัทใน terminal — แผนก, role, skill ของแต่ละ role
npm run ask "..."ถามผ่าน command line ไม่ต้องเปิดเบราว์เซอร์
npm run validateตรวจว่า roster ถูกต้อง (role/skill/แผนก ครบไหม)
npm testรันชุดทดสอบทั้งหมด (validate + hire + memory + chat UI)

ตัวอย่างถามผ่าน command line:

npm run ask "ขอไอเดียชื่อร้านกาแฟ 5 ชื่อ"copy

3เช็คว่าตอนนี้มี role / skill อะไรบ้าง

npm run statuscopy

โชว์ทุกแผนก ทุก role พร้อม skill ของแต่ละตัว และ tag [hired by ...] สำหรับคนที่จ้างมาทีหลัง — role ที่จ้างผ่านแชทจะ อยู่ถาวร เห็นได้จากคำสั่งนี้

อาการแก้ยังไง
